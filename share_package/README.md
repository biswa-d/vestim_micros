# Complete VESTim Installation Package

This package contains VESTim, a GPU-accelerated battery state estimation tool.

## For Windows Users (No Linux Knowledge Required)

1. Extract this ZIP file to a folder on your computer
2. Open the `windows_installer` folder
3. Double-click `Install-VESTim.bat`
4. Follow the on-screen instructions

For more details, see the README.md file in the windows_installer folder.

## For Linux Users or Advanced Users

If you're comfortable with Linux commands, you can:

1. Install the package directly with pip:
   ```
   pip install vestim-reg-gpu-1.0.0.tar.gz
   ```

2. Run the application:
   ```
   vestim-gui
   ```

For detailed instructions, see WINDOWS_REMOTE_GUIDE.md (for Windows users)
or README.md (for general information).
